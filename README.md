## Configuration

- Create file `KeyConfig.xcconfig` and add the keys `IMGUR_KEY`, `LOL_KEY`, `APNS_KEY`, `SHACK_ERUSER`, and `SHACK_ERPASS`.

- Create a file for bad words in `Resourices\Data` called `Words.json`.
```
{
    "words": [
        { "word": "word1" },
        { "word": "word2" },
        { "word": "word3" }
    ]
}
```
